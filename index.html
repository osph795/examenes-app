Configurar maestro

Profesor

URL del Apps Script maestro

El maestro es el Apps Script que gestiona la tabla central de alumnos (nombre, PIN y asignaturas a las que tiene acceso cada uno). Se usa tanto por el profesor (para crear/editar alumnos) como por los alumnos (para verificar su PIN).

ℹ️ Cómo obtener la URL

En el Google Sheet del maestro (el que tiene la pestaña alumnos): Extensiones → Apps Script → Implementar → Gestionar implementaciones → copia la URL de aplicación web.
